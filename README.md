## 使用方法

1. 使用可以安装插件的浏览器(如[Kiwi Browser](https://kiwibrowser.com/))+脚本管理器(如[Tampermonkey](https://www.tampermonkey.net/))，或使用可以直接使用脚本的浏览器(如[Via](https://viayoo.com/zh-cn/))
2. 转到Sleazy Fork的脚本页面：[E-Hentai移动端界面适配]()
3. 点击“安装此脚本”按钮
4. 在弹出窗口点击“安装”按钮
5. 完成。
